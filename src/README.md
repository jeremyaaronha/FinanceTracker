# Overview

As a software engineer, I want to continue improving my understanding of programming fundamentals and object-oriented programming. In this project, I created a console-based Finance Tracker using Kotlin to practice building a complete and functional application.

This software allows users to add income and expense transactions, view all transactions, calculate the current balance, and delete transactions by ID. The program uses classes, a data class, collections, loops, conditionals, and the `when` statement.

The purpose of writing this software was to better understand Kotlin syntax, especially how Kotlin handles classes, mutable and immutable variables, collections, and control structures. I also wanted to practice organizing code in a clean and simple way.

# Development Environment

I used IntelliJ IDEA to write and run the program.

The programming language used in this project is Kotlin. The project runs on the Java Virtual Machine (JVM) using the Eclipse Temurin JDK. I used Kotlin standard libraries such as mutable lists and Scanner for user input.
